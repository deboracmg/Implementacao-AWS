# Farmácia em Nuvem - Projeto AWS

Este projeto consiste na implementação de uma plataforma em nuvem para uma farmácia fictícia, utilizando serviços da AWS de forma estratégica para **reduzir custos**, **gerenciar dados de forma eficiente** e **garantir escalabilidade** da aplicação conforme a demanda da empresa.
